## MWP:Maa周管理

需要在MWP目录自行创建MaaConnect.json
内容为：

```json
{
    "Maa_core": "Maa本体所在文件夹绝对路径，字符串",
    "Adb": "模拟器adb绝对路径，字符串",
    "Address": "127.0.0.1:模拟器adb地址，字符串"
}
```
